# ai-translator-pro# 🌍 AI Translator Pro

AI Translator Pro is a futuristic AI-powered translator web application built using Flask, JavaScript, HTML, and CSS.

## 🚀 Features

- Real-time language translation
- Voice input support
- Text-to-speech output
- Translation history
- Modern futuristic UI
- Responsive dashboard design

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Deep Translator API
